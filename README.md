**Freshly** is a complete Laundry Management System designed to streamline laundry operations for customers, laundries, and delivery personnel. It provides an intuitive interface for placing and managing laundry orders, tracking order status in real time, and handling laundry items and services efficiently.

## Key Features
- **Order Management:** Create, update, and track laundry orders seamlessly.
- **Laundry Profiles:** Manage laundry items, services, and pricing.
- **Customer Dashboard:** View order history, current status, and available services.
- **Real-time Updates:** Notifications and status tracking for both laundries and customers.

## Technologies Used
- **Frontend:** Angular, Tailwind CSS
- **Backend:** ASP.NET Core
- **Database:** MySQL
- **Storage:** AWS S3 for image storage
